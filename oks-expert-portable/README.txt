# 🏗️ ОКС ЭКСПЕРТ - Система управления объектами

## 📦 Что это такое?

Это **единый файл** с системой для специалистов отдела капитального строительства и эксплуатации. 

**Ничего не нужно устанавливать!** Просто запустите файл и работайте.

---

## 🚀 Как запустить (3 способа)

### Способ 1: Двойной клик по установщику
1. Откройте папку `oks-expert-portable`
2. Дважды кликните на файл **`Установить.bat`**
3. Ярлык появится на рабочем столе
4. Приложение откроется автоматически

### Способ 2: Прямой запуск
1. Откройте папку `oks-expert-portable`
2. Дважды кликните на файл **`index.html`**
3. Приложение откроется в браузере

### Способ 3: Перетаскивание
1. Откройте любой браузер (Chrome, Firefox, Edge, Яндекс)
2. Перетащите файл `index.html` в окно браузера

## ✨ Что внутри?

### 📊 Панель мониторинга
- Статистика по всем объектам
- Открытые дефекты и проблемы
- Просроченные задачи
- Истекающие гарантии

### 🗺️ Карта объектов
- Все объекты Москвы и Московской области
- Цветовая индикация статусов:
  - 🔵 **Синий** - Строится
  - 🟢 **Зелёный** - Сдан
  - 🟠 **Оранжевый** - На гарантии
  - 🟣 **Фиолетовый** - Эксплуатация
  - 🔴 **Красный** - Аварийный

### 🏢 Реестр объектов
- Полный список объектов
- Фильтрация по статусам
- Быстрый доступ к деталям

### ⚠️ Журнал дефектов
- Регистрация проблем
- Приоритеты (критический, высокий, средний, низкий)
- Контроль устранения

### ✅ Задачи
- Назначение работ
- Контроль сроков
- Напоминания

### 📝 Полевые отчёты
- Фото/видео фиксация
- Геолокация
- Голосовые заметки
- Генерация актов

## 💡 Как работать

1. **Дашборд** - общая картина по всем объектам
2. **Карта** - визуальное отображение объектов
3. **Объекты** - подробная информация по каждому
4. **Дефекты** - регистрация и контроль проблем
5. **Задачи** - назначение и отслеживание работ
6. **Отчёты** - создание полевых отчётов с фото
