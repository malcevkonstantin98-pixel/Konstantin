# ОКС Эксперт - Упрощённая версия

## 🚀 Запуск за 1 минуту

### Способ 1: Просто откройте файл
1. Найдите файл `index.html` в этой папке
2. Дважды кликните по нему
3. Откроется в браузере - готово!

### Способ 2: Через браузер
1. Откройте любой браузер (Chrome, Firefox, Edge, Safari)
2. Нажмите `Ctrl+O` (или `Cmd+O` на Mac)
3. Выберите файл `index.html`

## ✨ Что работает сразу

- **Карта Москвы и МО** с объектами строительства
- **Дашборд** со статистикой
- **Полевые отчёты** с фото и геолокацией
- **Задачи и регламенты**
- **Карточки объектов** с документацией
- **Смена ролей** (администратор, инженер, подрядчик)

## 📱 Работа на телефоне/планшете

Файл адаптирован под мобильные устройства:
- Откройте файл в браузере телефона
- Все функции доступны сенсорным управлением
- Боковая панель сворачивается для экономии места

## ⚠️ Важное примечание

Для работы карты нужен API ключ Яндекс.Карт. 
В демо-режиме карта может отображаться с ограничениями.

Для полноценной работы:
1. Получите бесплатный ключ в [Кабинете разработчика Яндекс](https://developer.tech.yandex.ru/)
2. Откройте `index.html` в текстовом редакторе
3. Найдите строку: `apikey=your_api_key`
4. Замените `your_api_key` на ваш ключ

## 🎯 Основные возможности

### Дашборд
- Всего объектов: 8 (Москва и МО)
- Открытые дефекты: 62
- Просроченные задачи: отслеживаются
- Истекающая гарантия: 2 объекта

### Карта
- Все объекты на карте
- Цветовая индикация статусов
- Фильтрация по типу и статусу
- Клик по объекту → подробная информация

### Полевые отчёты
- Выбор объекта
- Тип проблемы (дефект, ТО, осмотр, авария)
- Фото/видео фиксация
- Геолокация автоматически
- Генерация QR-кода

### Задачи
- Список всех задач
- Приоритеты (критический, высокий, средний, низкий)
- Ответственные подрядчики
- Сроки выполнения

## 🔧 Технические детали

Используемые технологии:
- Vue 3 (через CDN)
- Tailwind CSS (стили)
- Font Awesome (иконки)
- Yandex Maps API v3 (карта)

Никакой установки не требуется - всё работает из одного файла!

## 📞 Поддержка

Если что-то не работает:
1. Проверьте подключение к интернету (нужен для загрузки библиотек)
2. Попробуйте другой браузер
3. Очистите кэш браузера
